Pompa de Caldura Clivet SPHERA EVO 2.0 BOX Split R-32

Unitate interna compacta de perete + unitate externa MiSAN-YEE · 2.1 - 8.1

Descriere

Sphera EVO 2.0 Box este sistemul split aer-apa Clivet cu unitate interioara compacta, montata pe perete. Se completeaza cu unitatea externa MiSAN-YEE (R-32). Pentru apa calda menajera se poate cupla cu un boiler extern ACS.

Compacta si eficienta, cu COP de peste 5 si functionare pana la -25°C. Disponibila si in varianta trifazica la marimile mari.

Caracteristici cheie

  • Unitate interioara compacta de perete;
  • COP > 5, clasa pana la A+++;
  • Cuplare cu boiler extern ACS (200-500L) pentru ACM;
  • Disponibil monofazic si trifazic (6.1-8.1);
  • Functioneaza pana la -25°C exterior;
  • Eligibil subventii pompe de caldura.
