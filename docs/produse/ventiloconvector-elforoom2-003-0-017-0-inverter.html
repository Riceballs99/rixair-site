Descriere

ELFORoom2 este ventiloconvectorul compact Clivet cu motor inverter, dedicat spatiilor rezidentiale mici. Solutie eficienta si silentioasa pentru incalzire si racire de la o pompa de caldura sau chiller.

Compact si discret, potrivit pentru camere si birouri mici, cu consum redus datorita motorului inverter.

  • Incalzire si racire;
